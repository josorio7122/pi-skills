# pi-skills

Personal collection of pi skills.

## Install

```bash
pi install git:github.com/josorio7122/pi-skills
```

## Skills

| Skill | Description | Source |
|-------|-------------|--------|
| **agents-md** | Write concise, minimal AGENTS.md files for AI agents | [getsentry/skills](https://github.com/getsentry/skills) |
| **docker-model-runner** | Run AI models locally with Docker Model Runner | [Docker](https://docs.docker.com/reference/cli/docker/model/skills/) |
| **exa-search** | Web search & content extraction via Exa API | [pi-packages](https://github.com/josorio7122/pi-packages) |
| **frontend-design** | Production-grade frontend interfaces | [anthropics/skills](https://github.com/anthropics/skills) |
| **gh** | GitHub CLI workflows | [pi-packages](https://github.com/josorio7122/pi-packages) |
| **glab** | GitLab CLI workflows | [pi-packages](https://github.com/josorio7122/pi-packages) |
| **interface-design** | Dashboard & app interface design | [Dammyjay93/interface-design](https://github.com/Dammyjay93/interface-design) |
| **pdf** | Read, create, and review PDF files | [openai/skills](https://github.com/openai/skills) |
| **playwright** | Browser automation via Playwright CLI | [openai/skills](https://github.com/openai/skills) |
| **render-deploy** | Deploy applications to Render | [openai/skills](https://github.com/openai/skills) |
| **skill-scanner** | Security audit for agent skills | [getsentry/skills](https://github.com/getsentry/skills) |
| **skill-writer** | Create, synthesize, and improve agent skills | [getsentry/skills](https://github.com/getsentry/skills) |

## Setup

exa-search requires the `EXA_API_KEY` environment variable:

```bash
export EXA_API_KEY=your-key-here
```
